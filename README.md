# CodeAlpha_LanguageTranslation-Tool

Task 1: Language Translation Tool

###  Description
A simple, easy-to-use **Language Translation Tool** built with Python. 
The app allows users to enter text, select a source and target language, 
and get an instant translation using the Google Translate API (via the 
`deep-translator` library).

### Features
- Simple user interface to enter text and choose languages
- Select source language and target language (10+ languages supported)
- Real-time translation using Google Translate API
- Displays translated text clearly on screen
- **Bonus:** Text-to-Speech - listen to the translated text out loud

###  Tech Stack
- Python 3
- deep-translator (Google Translate API wrapper)
- gTTS (Google Text-to-Speech)

###  How It Works
1. Enter the text you want to translate.
2. Choose the source language (or auto-detect).
3. Choose the target language you want to translate into.
4. View the translated text.
5. Optionally, listen to the translation using text-to-speech.
